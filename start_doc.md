# Waterfall API
1. Client sends JSON object with location info
2. API retrieves nearest waterfalls
3. API builds JSON object with one (or more) waterfall objects
4. API returns it to client

# What is in a Waterfall Object
Name of waterfall
Country
State/region
Nearest city
Coordinates (lat/lon)
Park/Reserve
River
Photo URLs
